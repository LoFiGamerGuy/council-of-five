# Changelog

All notable changes to the Council of Five methodology and templates.

The methodology itself is stable; specific examples, persona prompts, and operational tooling will evolve.

## [1.0.0] — 2026-05-07

### Added — initial public release
- `README.md` — what this is, why it exists, quick start, when not to use it
- `COUNCIL.md` — canonical methodology (load-bearing)
- `CHAIR-DISCIPLINE.md` — chair handoff principles
- `LICENSE` — CC BY 4.0 with author attribution
- `CONTRIBUTING.md` — what's welcome, what will be declined, style conventions
- `personas/` — Chair template plus 5 voice templates (Heimdall, Susanoo, Hephaestus, Thoth, Saraswati) as Claude Code subagent definitions
- `templates/` — convening log, kickoff prompt, cross-domain commitments
- `docs/` — speaking-order rationale, modes deep-dive, tiebreakers, failure modes, design lore
- `examples/example-convening.md` — one fully-worked example end-to-end

### Notes on this release
- The methodology in this release is distilled from ~12 months of internal use across multiple projects. Project-specific examples have been generalized; the structural patterns are unchanged.
- The full chair discipline (registry management, falsification clocks, amendment protocols) is workflow-specific and develops over time. `CHAIR-DISCIPLINE.md` covers the load-bearing principles; the operational details are for you to evolve.
